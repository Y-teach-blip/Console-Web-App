The Web Console: Logs information associated with a web page: network requests, JavaScript, CSS, security errors and warnings as well as error, warning and informational messages explicitly logged by JavaScript code running in the page context.

Why console is used:

Console is a basic computer or monitor and keyboard that is connected to another computer, server, or a mainframe over a network. It is used to maintain or monitor the status of the network or computer.

How to open console:

To open console right click on the screen and then in click inspect and then a window will open in left side that is your console.
